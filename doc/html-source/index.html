Welcome to the LIRC 0.9.1-git Manual

This manual will guide you through the installation and configuration of LIRC. Setting up LIRC currently is not an easy task. So please take some time to read this documentation carefully. If you don't understand a part of this manual or if you think there is something important missing you should complain.
